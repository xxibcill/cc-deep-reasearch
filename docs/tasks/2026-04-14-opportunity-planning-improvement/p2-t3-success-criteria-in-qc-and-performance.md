# Task P2-T3: Success Criteria In QC And Performance

## Objective

Carry opportunity-stage success criteria into QC review and post-publish evaluation so the system measures results against the original intent.

## Scope

- Feed `success_criteria` into QC prompts or checks.
- Feed the same criteria into performance-analysis framing.
- Make the comparison visible to operators.

## Acceptance Criteria

- QC can reference success criteria when evaluating readiness or missing elements.
- Performance analysis can compare results against the original brief.
- Operators can see whether a run matched the planned success definition.

## Advice For The Smaller Coding Agent

- Keep success-criteria usage explicit and inspectable.
- Avoid treating loosely worded criteria as hard metrics without normalization.
