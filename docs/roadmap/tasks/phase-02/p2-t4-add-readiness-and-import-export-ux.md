# P2-T4 - Add Readiness And Import Export UX

## Summary

Make strategy save flows safer and more informative with readiness feedback and advanced import/export support.

## Scope

- Add readiness panels showing missing required fields and warnings.
- Keep JSON import/export available as advanced tools.
- Improve save feedback, dirty state handling, and destructive-action confirmation.

## Deliverables

- Strategy readiness UI
- Advanced import/export controls
- Safer save state UX

## Dependencies

- Phase 01 validation data availability
- P2-T1 workspace layout

## Acceptance Criteria

- Operators can see why a strategy is incomplete before saving or using it.
- Raw JSON import/export remains possible without being the main workflow.
