# P3-T1 - Wire Strategy Into Pipeline Stages

## Summary

Use the richer strategy model to guide planning, scoring, scripting, and packaging decisions across the content pipeline.

## Scope

- Update prompt builders and agents to read the upgraded strategy sections.
- Route pillar, proof, audience-universe, platform, and CTA policy into the stages where they matter.
- Audit orchestrator decisions that should consider strategy readiness or strategy rules.

## Deliverables

- Updated prompt builders and agent integrations
- Integration tests covering strategy influence in key stages
- Documentation of which fields steer which stages

## Dependencies

- Phase 01 schema foundation
- Strategy data populated through Phase 02 UI

## Acceptance Criteria

- Strategy fields materially affect planning and generation behavior.
- Field usage across stages is intentional and traceable.
